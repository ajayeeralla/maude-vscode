# Maude VSCode Extension

A [Maude](https://en.wikipedia.org/wiki/Maude_system) syntax highlighting extension for VSCode

## Features

* Syntax highlighting
* Bracket matching
* Bracket autoclosing
* Bracket autosurrounding
* Comment toggling
* Auto indentation
